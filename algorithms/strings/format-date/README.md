- **カテゴリ**: 文字列 / フォーマット

## 問題概要

年 `y`、月 `m`、日 `d` がスペース区切りで与えられる。`"/"` 区切りの文字列にして出力する。

## 制約

- 2000 ≦ y ≦ 2016、1 ≦ m ≦ 12、1 ≦ d ≦ 31（実在しない日付の可能性あり）

## 入出力例

入力:
```
2019 3 5
```
出力:
```
2019/3/5
```

入力:
```
2020 11 8
```
出力:
```
2020/11/8
```

## 解法メモ

- 3 つの値をテンプレートリテラル `${y}/${m}/${d}` で連結する。計算量 O(1)。
- 数値計算が不要なので、区切り文字を替えるだけの `lines[0].split(" ").join("/")` でもよい。
