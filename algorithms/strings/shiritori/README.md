- **カテゴリ**: 標準入力 / 文字列

## 問題概要

1 行目に単語数 N、続く N 行に単語が与えられる。
連続する単語について「前の単語の末尾の文字 == 次の単語の先頭の文字」（大文字小文字を区別）が
すべて成り立てば `Yes`。成り立たない場合、最初に失敗した箇所の
「前の単語の末尾の文字 x」と「次の単語の先頭の文字 y」を `x y` の形式で出力する。

## 制約

- 2 ≦ N ≦ 1000、1 ≦ 各単語の長さ ≦ 50、英字（大小）

## 入出力例

| 入力 | 出力 |
| --- | --- |
| `3` / `apple` / `eagle` / `echo` | `Yes` |
| `3` / `cat` / `tiger` / `dog` | `r d` |

（例 2 は `tiger` の末尾 `r` と `dog` の先頭 `d` が不一致）

## 解法メモ

- 隣り合う単語のペアを走査し、`words[i].slice(-1)` と `words[i + 1][0]` を比較。
- 最初の不一致で `x y` を返し、最後まで一致すれば `Yes`。計算量 O(N)。
