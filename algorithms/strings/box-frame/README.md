- **カテゴリ**: 標準入力 / 文字列

## 問題概要

文字列 S を `+` の枠で囲んで出力する。上下に S の長さ + 2 個の `+`、
中央は S の左右を `+` で挟んだ行。

## 制約

- S は英字のみ、1 ≦ |S| ≦ 100

## 入出力例

入力:
```
Go
```
出力:
```
++++
+Go+
++++
```

入力:
```
Test
```
出力:
```
++++++
+Test+
++++++
```

## 解法メモ

- 枠の幅は `S.length + 2`。上下は `"+".repeat(幅)`、中央は `` `+${S}+` ``。3 行を `join("\n")`。計算量 O(|S|)。
