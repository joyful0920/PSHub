- **カテゴリ**: 文字列 / 回転

## 問題概要

N 面の箱の各側面に 1 文字ずつ書かれている。目標の向き `t` と現在の向き `s` が与えられる。
箱は右回りにしか回せず、1 面分の回転を 1 回と数える。`s` を回転させて `t` に一致させる
最小回転数を出力する。

右に 1 回転すると、先頭の文字が末尾に移る（`"abcde"` → `"bcdea"`）。

## 制約

- 1 ≦ N ≦ 100、|t| = |s| = N、半角英字
- 必ず N 回以内で t を作れる

## 入出力例

入力:
```
5 abcde cdeab
```
出力:
```
3
```
（`cdeab` → `deabc` → `eabcd` → `abcde` で 3 回）

入力:
```
4 abab baba
```
出力:
```
1
```
（`baba` → `abab` で 1 回）

## 解法メモ

- `s` を右に `k` 回転させた文字列は `s.slice(k) + s.slice(0, k)`。
- `k = 0` から順に `t` と比較し、最初に一致した `k` が最小回転数。
- N ≦ 100 なので、各回転で文字列を作り直しても O(N^2) で十分間に合う。
