
- **カテゴリ**: 標準入力 / 文字列 / 正規表現

## 問題概要

英数字とハイフン `-` からなる文字列を受け取り、
長さ 2 以上のハイフンの連続をすべて長さ 1 のハイフンにして出力する。

## 制約

- 1 ≦ s の長さ ≦ 100、s は英数字またはハイフン

## 入出力例

| 入力 | 出力 |
| --- | --- |
| `--a--b---c-` | `-a-b-c-` |
| `foo---bar` | `foo-bar` |

## 解法メモ

- `replace(/-+/g, "-")`：`-+` がハイフンの連続（1 個以上）を貪欲に 1 つのまとまりとして捉え、
  `g` フラグで全箇所を `-` 1 つに置換する。計算量 O(n)。
