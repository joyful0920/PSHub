- **カテゴリ**: 標準入力 / 文字列

## 問題概要

`s n` の形式で、文字列 s と整数 n が空白区切りの 1 行で与えられる。
s の左から数えて n 文字目（1-index）を出力する。

## 制約

- 1 ≦ s の長さ ≦ 10、1 ≦ n ≦ s の長さ
- s は半角アルファベット

## 入出力例

| 入力 | 出力 |
| --- | --- |
| `hello 2` | `e` |
| `abcdef 4` | `d` |

## 解法メモ

- 空白で分割し、`str[n - 1]` を取る（1-index → 0-index）。計算量 O(1)。
