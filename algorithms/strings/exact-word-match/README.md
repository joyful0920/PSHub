- **カテゴリ**: 標準入力 / 文字列

## 問題概要

1 行目に探したい語 S、2 行目に単語数 N、3 行目に空白区切りの N 個の単語が与えられる。
S が単語リストに**完全一致**で含まれれば `Yes`、なければ `No` を出力する。
（部分一致は不可。例: `apple` は `pineapple` の一部だが一致とはみなさない）

## 制約

- 1 ≦ S の長さ ≦ 10、2 ≦ N ≦ 5、1 ≦ 各単語の長さ ≦ 10、英字小文字

## 入出力例

入力:
```
pizza
3
cheese pizza salad
```
出力:
```
Yes
```

入力:
```
egg
2
eggplant boiled
```
出力:
```
No
```
（`egg` は `eggplant` の一部だが、完全一致する単語ではないので `No`）

## 解法メモ

- 空白で分割した単語配列に対し `includes(S)` で完全一致判定。計算量 O(N)。
