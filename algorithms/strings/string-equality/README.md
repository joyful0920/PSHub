- **カテゴリ**: 標準入力 / 文字列

## 問題概要

2 つの文字列 S, T が改行区切りで与えられる（1 行目 S、2 行目 T）。
S と T が完全に一致すれば `Yes`、そうでなければ `No` を出力する。

## 制約

- 1 ≦ S, T の長さ ≦ 10、半角英数字

## 入出力例

| S | T | 出力 |
| --- | --- | --- |
| `apple` | `apple` | `Yes` |
| `sun` | `moon` | `No` |

## 解法メモ

- 長さを比べ、一致すれば 1 文字ずつ照合。全て一致で `Yes`。
- 簡潔版: `s1 === s2 ? "Yes" : "No"`（文字列は `===` で丸ごと比較できる）。計算量 O(n)。
