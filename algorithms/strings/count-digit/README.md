- **カテゴリ**: 文字列 / カウント

## 問題概要

数字 `n`（0〜9）と文字列 `s` が改行区切りで与えられる。`s` の中に `n` が何個含まれるかを出力する。

## 制約

- 0 ≦ n ≦ 9（整数）
- 1 ≦ s の長さ ≦ 100、s は半角英数字

## 入出力例

入力:
```
7
70707abc
```
出力:
```
3
```

入力:
```
1
abcdefg
```
出力:
```
0
```

## 解法メモ

- `n` を文字列のまま扱い、`s` を 1 文字ずつ走査して一致数を数える。計算量 O(|s|)。
- `[...s].filter(c => c === n).length` や `s.split(n).length - 1` でも同じ。
