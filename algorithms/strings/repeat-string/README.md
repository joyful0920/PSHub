- **カテゴリ**: 標準入力 / 文字列

## 問題概要

整数 N と文字列 S が改行区切りで与えられる。S を N 回繰り返して連結した文字列を出力する。

## 制約

- 1 ≦ N ≦ 100、1 ≦ |S| ≦ 10、S は英字小文字

## 入出力例

入力:
```
3
xy
```
出力:
```
xyxyxy
```

入力:
```
5
a
```
出力:
```
aaaaa
```

## 解法メモ

- 組み込みの `S.repeat(N)` で 1 行。ループ + `push` + `join` でも書けるが、意図がそのまま
  読める `repeat` が定石。計算量 O(N × |S|)。
