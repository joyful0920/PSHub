- **カテゴリ**: 文字列 / 集合（種類数）

## 問題概要

パスワード文字列 S が与えられる。**全ての文字が同じ**なら `NG`、そうでなければ `OK` を
出力する。

## 制約

- 2 ≦ S の長さ ≦ 100、S は半角英数字

## 入出力例

入力:
```
zzzz
```
出力:
```
NG
```

入力:
```
zzza
```
出力:
```
OK
```

## 解法メモ

- 「全て同じ文字」=**文字の種類数が 1**。`new Set(s).size === 1` の 1 行で判定できる。
- 別解 1: `[...s].every((c) => c === s[0])` — 全要素が条件を満たすかは `every`。
- 別解 2: 先頭文字と順に比較して違いが見つかったら早期 break（O(|S|) で最速だが行数は増える）。
- 計算量はいずれも O(|S|)。
