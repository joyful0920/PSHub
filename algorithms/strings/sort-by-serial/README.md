- **カテゴリ**: 文字列 / ソート

## 問題概要

ユーザー ID は「ユーザーネーム（英小文字）+ 通し番号（登録順の整数）」の形。N 個の ID が
与えられるので、末尾の通し番号が小さい順（登録の早い順）に並べて出力する。

## 制約

- 1 ≦ N ≦ 200、ユーザーネームは英小文字のみ 1〜20 文字
- 通し番号は 1〜10000、先頭に不要な 0 なし、重複なし

## 入出力例

入力:
```
3
node12
vue3
react256
```
出力:
```
vue3
node12
react256
```
（3 < 12 < 256）

入力:
```
2
widget99
gadget5
```
出力:
```
gadget5
widget99
```

## 解法メモ

- 末尾の通し番号を正規表現 `/\d+$/`（末尾に続く数字列）で取り出し、数値化してソートキーにする。
- 数値ソートは比較関数を渡す：`sort((a, b) => serial(a) - serial(b))`。文字列ソートのままだと
  `"1000" < "2"` のように辞書順になってしまうので注意。
- 「通し番号を取り出す」を `serial` というヘルパー関数に切り出すと意図が明確。計算量 O(N log N)。
