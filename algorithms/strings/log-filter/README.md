- **カテゴリ**: 標準入力 / 文字列 / フィルタ

## 問題概要

1 行目に件数 N、2 行目にキーワード G、続く N 行にログ文字列が与えられる。
キーワード G を含むログだけを入力順に出力する。1 つも含まれない場合は `None` を出力する。

## 制約

- 1 ≦ N ≦ 100、1 ≦ 各文字列の長さ ≦ 100、半角英小文字

## 入出力例

入力:
```
3
err
error
info
errno
```
出力:
```
error
errno
```

入力（該当なし）:
```
2
zzz
abc
def
```
出力:
```
None
```

## 解法メモ

- 各ログについて `includes(G)` で部分一致を判定し、一致したものを出力。
- 一致が 1 つも無ければ `None`。計算量 O(N × 文字列長)。
