- **カテゴリ**: 標準入力 / 幾何・面積

## 問題概要

H × W の画面をスクロールする。前回の描画結果のうち重なる部分はコピーで再利用でき、
新しく描く画素だけを描画する。縦 dy・横 dx スクロールしたとき、描画する画素数を出力する。
dy, dx は負になることもある。

## 制約

- 1 ≦ H, W ≦ 10,000、-H ≦ dy ≦ H、-W ≦ dx ≦ W

## 入出力例

入力:
```
10 10
2 3
```
出力:
```
44
```
（全体 100 − 再利用できる重なり (10-2)×(10-3)=56 → 44）

入力:
```
8 6
-8 6
```
出力:
```
48
```
（スクロール量が画面と同じで重なりが 0 → 全画面 48 を描画）

## 解法メモ

- 描画量 = 「画面全体 `H*W`」 −「コピーで再利用できる重なり矩形」。
- 重なり矩形は縦 `H - |dy|`、横 `W - |dx|`。スクロールが負でも大きさは絶対値なので `Math.abs`。
- L 字領域を直接足すより、全体から引く方が項が少なく明快。計算量 O(1)。
