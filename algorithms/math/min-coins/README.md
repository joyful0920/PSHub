- **カテゴリ**: 標準入力 / 貪欲法

## 問題概要

金額 X 円をちょうど支払うとき、使う硬貨（500・100・50・10・5・1 円）の
合計枚数を最小にした場合の枚数を出力する。

## 制約

- 1 ≦ X < 1000（整数）

## 入出力例

| 入力 | 出力 |
| --- | --- |
| `99` | `10` |
| `63` | `5` |

（例: 99 = 50×1 + 10×4 + 5×1 + 1×4 で 10 枚）

## 解法メモ

- 高額の硬貨から順に `floor(remaining / coin)` 枚使い、余りを `remaining %= coin` で更新。
- この硬貨体系では貪欲法で最小枚数になる。計算量 O(硬貨種類数)。
