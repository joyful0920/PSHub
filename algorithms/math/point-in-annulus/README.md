- **カテゴリ**: 標準入力 / 幾何

## 問題概要

中心 (xc, yc)、半径 r_1 < r_2 の 2 つの同心円が与えられる。
続いて n 人の座標が与えられ、各人が 2 つの円の間（r_1 以上 r_2 以下の距離）に
いれば `yes`、そうでなければ `no` を出力する。

## 制約

- 1 ≦ n ≦ 100、1 ≦ r_1 < r_2 ≦ 100、座標は -100〜100 の整数

## 入出力例

入力:
```
0 0 2 5
3
0 0
3 0
6 0
```
出力:
```
no
yes
no
```

## 解法メモ

- 中心からの距離の 2 乗 `(x-xc)^2 + (y-yc)^2` を `r_1^2` と `r_2^2` と比較。
- 平方根を取らず 2 乗のまま比較 → 整数演算で誤差なし。計算量 O(n)。
