- **カテゴリ**: 標準入力 / 集計（平均 + 切り上げ）

## 問題概要

N 個の箱の水深が与えられる。その平均を求め、**小数点以下を切り上げて**出力する。

## 入力フォーマット

```
N
d_1 d_2 ... d_N
```

## 制約

- 1 ≦ N ≦ 100、1 ≦ d_i ≦ 1000

## 入出力例

入力:
```
4
1 2 2 2
```
出力:
```
2
```
（合計 7 / 4 = 1.75 → 切り上げて 2）

入力:
```
3
10 10 10
```
出力:
```
10
```

## 解法メモ

- `reduce` で合計を求め、`n` で割って平均、`Math.ceil` で切り上げ。計算量 O(N)。
- 切り上げ＝`Math.ceil`（切り捨ては `Math.floor`、四捨五入は `Math.round`）。
