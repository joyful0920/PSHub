- **カテゴリ**: 標準入力 / 算術（切り上げ除算）

## 問題概要

計算にかかる時間 T（時間）が与えられる。何日で計算が終わるかを出力する。
ちょうど割り切れないときは、終わっている最小の日数（切り上げ）を出す。

例: 40 時間 = 1 日と 16 時間 → 2 日。

## 制約

- 1 ≦ T ≦ 100

## 入出力例

入力:
```
25
```
出力:
```
2
```
（24 時間を超えるので 2 日）

入力:
```
72
```
出力:
```
3
```
（ちょうど 3 日）

## 解法メモ

- 1 日 = 24 時間。端数があれば繰り上げるので切り上げ除算 `Math.ceil(T / 24)`。計算量 O(1)。
- `Math.floor(T / 24) + (T % 24 ? 1 : 0)` と手動で書いても同じ。
