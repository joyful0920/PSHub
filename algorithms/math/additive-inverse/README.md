- **カテゴリ**: 標準入力 / 算術

## 問題概要

整数 `n` が与えられる。`n` に足すと 0 になる数（符号を反転した数）を出力する。

## 制約

- -10 ≦ n ≦ 10（整数）

## 入出力例

入力:
```
-4
```
出力:
```
4
```

入力:
```
0
```
出力:
```
0
```

## 解法メモ

- `0 - n` で符号を反転する。計算量 O(1)。
- `-n` だと `n === 0` のとき `-0` になり、`console.log(-0)` は `"-0"` を出力してしまう。`0 - n` は `0` を返すのでこの問題を避けられる。
