- **カテゴリ**: 標準入力 / 算術

## 問題概要

1 行にスペース区切りで 2 つの整数 `N M` が与えられる。片方が偶数・片方が奇数の組み合わせなら
`YES`、そうでなければ `NO` を出力する。

## 制約

- 1 ≦ N, M ≦ 50

## 入出力例

入力:
```
4 7
```
出力:
```
YES
```

入力:
```
6 8
```
出力:
```
NO
```

## 解法メモ

- 偶数 + 奇数の和は必ず奇数。よって `(N + M) % 2 === 1` なら偶奇のペア。計算量 O(1)。
- `N % 2 !== M % 2` でも同じ判定になる。
