- **カテゴリ**: 標準入力 / 数論 / 約数

## 問題概要

N の約数のうち N 自身を除いた和を S とする。Q 個の整数それぞれについて判定する。

- N = S なら `perfect`（完全数）
- |N - S| = 1 なら `nearly`（ほぼ完全数）
- どちらでもなければ `neither`

## 制約

- 1 ≦ Q ≦ 50、2 ≦ N ≦ 1000

## 入出力例

入力:
```
3
28
16
7
```
出力:
```
perfect
nearly
neither
```
（28 = 1+2+4+7+14 完全 / 16: 1+2+4+8=15 で |16-15|=1 / 7: 1 のみで neither）

入力:
```
2
6
9
```
出力:
```
perfect
neither
```
（6 = 1+2+3 完全 / 9: 1+3=4 で neither）

## 解法メモ

- 各 N について 1〜N-1 の約数を足して真約数和 S を求める。
- `S === N` で perfect、`Math.abs(N - S) === 1` で nearly（定義の |N-S|=1 に忠実）。計算量 O(Q × N)。
