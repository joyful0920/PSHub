- **カテゴリ**: 標準入力 / 数論 / 約数

## 問題概要

N の約数のうち N 自身を除いた和を S とする。Q 個の整数それぞれについて判定する。

- N = S なら `perfect`（完全数）
- |N - S| = 1 なら `nearly`（ほぼ完全数）
- どちらでもなければ `neither`

## 制約

- 1 ≦ Q ≦ 50、2 ≦ N ≦ 1000

## 入出力例

入力:
```
3
6
8
12
```
出力:
```
perfect
nearly
neither
```
（6 = 1+2+3 完全 / 8: 1+2+4=7 で |8-7|=1 / 12: 1+2+3+4+6=16）

入力:
```
2
496
10
```
出力:
```
perfect
neither
```

## 解法メモ

- 各 N について 1〜N-1 の約数を足して真約数和 S を求める。
- `S === N` で perfect、`Math.abs(N - S) === 1` で nearly（定義の |N-S|=1 に忠実）。計算量 O(Q × N)。
