- **カテゴリ**: 標準入力 / 集計

## 問題概要

複数の整数が 1 行に 1 つずつ与えられる。最大値を 1 行目、最小値を 2 行目に出力する。

## 制約

- 入力される整数は 1 個以上

## 入出力例

入力:
```
5
8
2
7
4
```
出力:
```
8
2
```

入力:
```
10
10
10
```
出力:
```
10
10
```

## 解法メモ

- `Math.max(...numbers)` / `Math.min(...numbers)` で集計。計算量 O(N)。
- スプレッド展開は要素数が極端に多いとスタック上限に触れうるが、この規模では問題なし。大量要素なら `reduce` で畳み込む。
