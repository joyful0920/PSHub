- **カテゴリ**: 標準入力 / 算術

## 問題概要

和暦（元号 + 年）が与えられるので西暦に変換して出力する。元号は `"S"`（昭和, 1926 年開始）
または `"H"`（平成, 1989 年開始）。1 行目に元号、2 行目に年 N が与えられる。

## 制約

- 元号は "S" または "H"
- "S" のとき 1 ≦ N ≦ 64、"H" のとき 1 ≦ N ≦ 31

## 入出力例

入力:
```
S
50
```
出力:
```
1975
```
（昭和 50 年 = 1926 + 49）

入力:
```
H
10
```
出力:
```
1998
```
（平成 10 年 = 1989 + 9）

## 解法メモ

- 元年（1 年）が開始年に当たるので、`西暦 = 開始年 + (N - 1)`。
  昭和は 1926、平成は 1989 を基準にする。計算量 O(1)。
- off-by-one に注意（`+ N` ではなく `+ (N - 1)`）。
