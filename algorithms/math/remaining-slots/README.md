- **カテゴリ**: 標準入力 / 算術

## 問題概要

画面を N × N のマスに分割し、1 マスに 1 人が映る。現在 M 人が出席しているとき、
あと何人参加できるかを出力する。

## 制約

- 1 ≦ N ≦ 10、0 ≦ M ≦ N × N

## 入出力例

入力:
```
6
10
```
出力:
```
26
```
（6×6 = 36 マス、出席 10 人 → 残り 26）

入力:
```
4
5
```
出力:
```
11
```
（4×4 = 16 マス、出席 5 人 → 残り 11）

## 解法メモ

- マス総数 `N * N` から出席者 `M` を引くだけ。計算量 O(1)。
