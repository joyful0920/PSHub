- **カテゴリ**: 標準入力 / 算術

## 問題概要

画面を N × N のマスに分割し、1 マスに 1 人が映る。現在 M 人が出席しているとき、
あと何人参加できるかを出力する。

## 制約

- 1 ≦ N ≦ 10、0 ≦ M ≦ N × N

## 入出力例

入力:
```
5
10
```
出力:
```
15
```
（5×5 = 25 マス、出席 10 人 → 残り 15）

入力:
```
2
4
```
出力:
```
0
```
（2×2 = 4 マス、すでに満席）

## 解法メモ

- マス総数 `N * N` から出席者 `M` を引くだけ。計算量 O(1)。
