- **カテゴリ**: 標準入力 / 幾何 / 集計

## 問題概要

1 辺 D の正方形の紙を N 枚、左右に連結して 1 枚の垂れ幕を作る。
2 枚目以降は 1 つ前の紙と d_i だけ重ねて貼る。完成した垂れ幕の面積を求める。

## 制約

- 2 ≦ N ≦ 100、2 ≦ D ≦ 10、1 ≦ d_i ≦ D/2

## 入出力例

入力:
```
2 10
3
```
出力:
```
170
```

入力:
```
3 5
2
1
```
出力:
```
60
```
（幅 = 5 + (5-2) + (5-1) = 12、面積 = 5 × 12 = 60）

## 解法メモ

- 全体の横幅は「最初の D + 2 枚目以降の (D - d_i) の総和」。面積 = 幅 × D。
- 同値な式: `N × D² − (Σ d_i) × D`。計算量 O(N)。
