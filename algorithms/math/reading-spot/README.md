- **カテゴリ**: 標準入力 / 幾何（距離判定）

## 問題概要

工事現場が座標 `(a, b)` にある。木陰が N 箇所あり、各木陰の座標が与えられる。
各木陰について、工事現場から **R メートル以上**離れていれば `silent`、そうでなければ
`noisy` を 1 行ずつ出力する。

判定条件: `√((x-a)² + (y-b)²) ≥ R`

## 入力フォーマット

```
a b R
N
x_i y_i   （N 行）
```

## 制約

- 0 ≦ a, b, x, y ≦ 100、1 ≦ R ≦ 100、1 ≦ N ≦ 1000（すべて整数）

## 入出力例

入力:
```
1 1 4
3
1 5
4 1
2 2
```
出力:
```
silent
noisy
noisy
```
（距離² = 16, 9, 2。R² = 16。16 ≥ 16 は silent、9 と 2 は noisy）

## 解法メモ

- **落とし穴回避**: `√` を取って `距離 >= R` と比べると浮動小数点の誤差が出る。
  **両辺を 2 乗して `(x-a)² + (y-b)² >= R²` で比較**すれば、すべて整数演算になり
  誤差ゼロ・境界値も正確。距離比較の定石。
- 「以上」なので R ちょうど（距離² = R²）は silent 側。計算量 O(N)。
