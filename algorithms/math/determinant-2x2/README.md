- **カテゴリ**: 標準入力 / 2 次元配列 / 算術

## 問題概要

2 行 2 列の行列が与えられる。

```
a b
c d
```

この 2 次行列の行列式 `a × d - b × c` を計算して出力する。

## 制約

- 各要素は整数、-1000 ≦ 各値 ≦ 1000

## 入出力例

入力:
```
2 5
1 3
```
出力:
```
1
```
（2×3 - 5×1 = 6 - 5 = 1）

入力:
```
4 4
2 2
```
出力:
```
0
```
（4×2 - 4×2 = 0）

## 解法メモ

- 各行を `split(" ").map(Number)` で数値化し、2 次元配列 `grid` にする。
- ネストした分割代入 `const [[a, b], [c, d]] = grid;` で 4 要素を取り出すと、
  式 `a * d - b * c` が問題文の定義とそのまま対応して読みやすい。計算量 O(1)。
