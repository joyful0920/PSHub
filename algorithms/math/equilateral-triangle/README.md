- **カテゴリ**: 標準入力 / 条件分岐

## 問題概要

三角形の 3 辺の長さ a, b, c が改行区切りで与えられる。
3 辺すべてが等しければ（正三角形なら）`YES`、そうでなければ `NO` を出力する。

## 制約

- 1 ≦ a, b, c ≦ 100（整数）

## 入出力例

| a | b | c | 出力 |
| --- | --- | --- | --- |
| `9` | `9` | `9` | `YES` |
| `2` | `3` | `4` | `NO` |

## 解法メモ

- `a === b && b === c` なら 3 辺すべて等しい（推移律で a === c も成立）。計算量 O(1)。
