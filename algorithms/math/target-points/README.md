- **カテゴリ**: 標準入力 / 集計

## 問題概要

会計 100 円につき 1 ポイントが貯まる（1 回の会計で `X` 円なら `floor(X / 100)` ポイント）。
目標は `M` ポイント。`N` 回分の会計記録が与えられるので、目標に到達するのにあと何円分
買い物すればよいかを出力する。すでに `M` ポイント貯まっていれば `0` を出力する。

## 制約

- 1 ≦ N ≦ 100、1 ≦ M ≦ 10,000、1 ≦ t_i ≦ 100,000

## 入出力例

入力:
```
2 100
2500 3000
```
出力:
```
4500
```
（獲得ポイント 25 + 30 = 55、残り 45 ポイント → 4500 円）

入力:
```
3 30
900 500 800
```
出力:
```
800
```
（獲得ポイント 9 + 5 + 8 = 22、残り 8 ポイント → 800 円）

## 解法メモ

- 各会計を `floor(t / 100)` でポイント化し `reduce` で合計。
- 残りポイント `M - sum` が正ならその分（× 100 円）が必要、0 以下なら不要。
  `Math.max(M - sum, 0) * 100` で両ケースをまとめられる。計算量 O(N)。
