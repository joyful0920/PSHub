- **カテゴリ**: 標準入力 / 倍数判定

## 問題概要

`N X Y` が空白区切りで与えられる。1〜N の各番号 i について、
X の倍数なら `A`、Y の倍数なら `B`、両方なら `AB`、どちらでもなければ `N` を
1 行ずつ出力する（計 N 行）。

## 制約

- 1 ≦ N ≦ 1000、1 ≦ X, Y ≦ N

## 入出力例

入力:
```
4 2 3
```
出力:
```
N
A
B
A
```

入力:
```
6 6 2
```
出力:
```
N
B
N
B
N
AB
```

## 解法メモ

- 各 i について `i % X === 0` なら A、`i % Y === 0` なら B を付け、空なら N。計算量 O(N)。
