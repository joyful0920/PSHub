- **カテゴリ**: 標準入力 / 進法変換（bijective base-26）

## 問題概要

表計算ソフトの列名ルール：左から `A, B, …, Z, AA, AB, …, ZZ, AAA, …`。
左から N 番目のマスの名前を出力する。

## 制約

- 1 ≦ N ≦ 18,278（= ZZZ、最大 3 文字）

## 入出力例

入力:
```
28
```
出力:
```
AB
```

入力:
```
55
```
出力:
```
BC
```

入力:
```
702
```
出力:
```
ZZ
```

## 解法メモ

- これは **bijective base-26**（0 に当たる文字がない 26 進法。A=1 … Z=26、Z の次が AA）。
  普通の 26 進法とズレるので、各桁を取り出す前に **`n -= 1` して 0-index に直す**のが鍵。
  - 例: n=26 → `n--`=25 → `25 % 26 = 25` = 'Z'、`25 / 26` 切り捨て = 0 で終了。
- `String.fromCharCode(65 + (n % 26))` で 0〜25 を A〜Z に変換し、低い桁から求まるので
  結果の**前に連結**。桁数は `while (n > 0)` で自動的に決まる（1〜3 文字を場合分け不要）。
  計算量 O(桁数)。
