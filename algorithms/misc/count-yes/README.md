- **カテゴリ**: 標準入力 / 文字列（カウント）

## 問題概要

`"y"` と `"n"` のみからなる長さ 10 の文字列 S が与えられる（`y` = 承諾、`n` = 不承諾）。
承諾された数、つまり `"y"` の個数を出力する。

## 制約

- S の長さ = 10、S は "y", "n" のみ

## 入出力例

入力:
```
yyyynnnnny
```
出力:
```
5
```

入力:
```
nnnnnnnnnn
```
出力:
```
0
```

## 解法メモ

- `"y"` の個数を数えるだけ。`for (const c of s)` で 1 文字ずつ（文字列はそのまま反復可能なので
  `[...s]` は不要）。計算量 O(|S|)。
- `[...s].filter(c => c === "y").length` や `s.split("y").length - 1` でも同じ。
