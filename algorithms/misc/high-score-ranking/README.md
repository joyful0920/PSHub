- **カテゴリ**: 標準入力 / 集計・ソート

## 問題概要

アイテムが N 種類あり、種類 i の単価が `C_i`。各ユーザーはアイテムを `X_i` 個持ち、
スコアは `Σ C_i × X_i`。小数第一位を四捨五入して整数にする。M 人分を計算し、
上位 K 件のスコアを高い順に出力する。

## 入力フォーマット

```
N M K
C_1 C_2 ... C_N     ← 単価（小数あり）
X_1 ... X_N          ← 各ユーザーの所持数（M 行）
```

## 制約

- 1 ≦ N ≦ 100、1 ≦ M ≦ 1000、1 ≦ K ≦ M
- 0 ≦ C_i ≦ 100、0 ≦ X_i ≦ 100000

## 入出力例

入力:
```
3 4 2
2 0.5 10
1 4 0
0 0 3
5 2 1
2 10 0
```
出力:
```
30
21
```
（スコア: 1・4・0→4, 0・0・3→30, 5・2・1→21, 2・10・0→9。上位 2 件は 30, 21）

## 解法メモ

- 各ユーザーのスコアは単価ベクトルとの**内積** `Σ C_i × X_i`。
- 四捨五入は `Math.round`（正の値なので小数第一位を half-up で正しく丸める）。
- 全員分を配列に入れ、降順ソートして `slice(0, k)`。計算量 O(M·N + M log M)。
- 内積は `counts.reduce((s, x, i) => s + x * points[i], 0)` とも書ける。
