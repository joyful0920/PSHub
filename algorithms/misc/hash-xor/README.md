- **カテゴリ**: 標準入力 / ビット演算（XOR）

## 問題概要

`"0"`/`"1"` からなる長さ N の文字列を、左から長さ M のブロックに分割する（最後のブロックは
長さ M になるよう `"0"` を追加）。各ブロックを左から順に、1 文字ずつ XOR していき、
最終的に得られる長さ M の文字列を出力する。

XOR: `0^0=0`, `1^1=0`, `0^1=1`, `1^0=1`（ビットが異なれば 1）。

## 入力フォーマット

```
N M
s
```

## 制約

- 1 ≦ N ≦ 1000、1 ≦ M ≦ 100、s の長さ = N、s は "0"/"1"

## 入出力例

入力:
```
8 3
10100101
```
出力:
```
110
```
（ブロック `101`, `001`, `01`→`010` を XOR: 101^001=100, 100^010=110）

入力:
```
2 5
10
```
出力:
```
10000
```
（ブロックが 1 つだけ・M より短くても M=5 に埋めて `10000`）

## 解法メモ

- 結果を長さ M の `0` 配列で初期化（XOR の単位元は 0 なので、最初のブロックはそのまま入る）。
- 各ブロックを `slice(i, i+m).padEnd(m, "0")` で **必ず M に揃える**。唯一のブロックが短い
  （N < M）ケースも padEnd で正しく埋まる。
- XOR は数値の `^` 演算子で `result[j] ^= Number(block[j])`。計算量 O(N)。
