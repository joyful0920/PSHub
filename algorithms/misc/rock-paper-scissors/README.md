- **カテゴリ**: 標準入力 / 条件分岐

## 問題概要

じゃんけんの対戦記録が N 行与えられる。各行は自分の手と相手の手が空白区切り
（`G`=グー, `C`=チョキ, `P`=パー）。自分が勝った回数を数えて出力する。
勝ち: G は C に、C は P に、P は G に勝つ。

## 制約

- 1 ≦ N ≦ 100、各手は `G` / `C` / `P`

## 入出力例

入力:
```
3
G C
C P
P G
```
出力:
```
3
```

入力:
```
2
C P
G G
```
出力:
```
1
```

## 解法メモ

- 各対戦で自分の手が相手の手に勝つかを判定し、勝ちを数える。計算量 O(N)。
