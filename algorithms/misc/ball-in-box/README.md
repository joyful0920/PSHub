- **カテゴリ**: 標準入力 / 条件抽出

## 問題概要

半径 `r` のボールと、n 個の箱（各箱は 高さ h・幅 w・奥行き d）が与えられる。
箱の**一番短い辺がボールの直径以上**ならそのボールを収納できる。収納できる箱の番号を
昇順にすべて出力する。

## 入力フォーマット

```
n r
h_i w_i d_i   （n 行）
```

## 制約

- 1 ≦ n ≦ 100、1 ≦ r ≦ 100、1 ≦ h, w, d ≦ 200

## 入出力例

入力:
```
3 2
3 5 5
4 4 4
6 6 6
```
出力:
```
2
3
```
（直径 = 4。箱1 は 3<4 で不可、箱2 は 4,4,4 で可、箱3 は 6,6,6 で可）

## 解法メモ

- **落とし穴**: `r` は**半径**。箱に入る条件は**直径 `2r` 以上**。半径のまま比べると誤り。
- 「一番短い辺が直径以上」＝「3 辺すべてが直径以上」。`Math.min(h,w,d) >= 2r` でも、
  `h>=2r && w>=2r && d>=2r` でも同じ。
- 直径はループ中変わらないので外で 1 回計算。計算量 O(n)。
