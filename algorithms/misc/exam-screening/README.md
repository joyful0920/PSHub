- **カテゴリ**: 標準入力 / 条件判定 / 集計

## 問題概要

N 人の受験者データが与えられる。各行は文理区分 `t`（理系 `s` / 文系 `l`）と 5 科目
（英・数・理・国・地）の点数。次の両方を満たす人数を数えて出力する。

- 5 科目の合計が 350 以上
- 理系(`s`) は 数+理 が 160 以上 / 文系(`l`) は 国+地 が 160 以上

## 制約

- 1 ≦ N ≦ 1000、各科目 0〜100、`t` は `s` または `l`

## 入出力例

入力:
```
3
s 80 90 90 50 50
s 100 100 100 10 10
l 80 80 80 60 60
```
出力:
```
1
```
（1 人目のみ通過。2 人目は合計 320 で不足、3 人目は文系 2 科目 120 で不足）

## 解法メモ

- 各行を分割し、区分 `t` と 5 点数に分ける。
- 合計 `total` と、区分に応じた 2 科目和 `grouped` を求め、両条件を満たせばカウント。計算量 O(N)。
