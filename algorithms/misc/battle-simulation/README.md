- **カテゴリ**: 標準入力 / シミュレーション

## 問題概要

レベル制の戦闘を N 回シミュレーションする。自分の初期レベル L と、対戦相手 N 人の
レベルが順に与えられる。各戦闘のルールは:

- レベルが高い方が勝ち。**勝者は相手のレベルの半分（切り捨て）だけ上昇**。
- **敗者は自分のレベルが半分（切り捨て）になる**。
- 同レベルは引き分けで変化なし。

N 回の戦闘後の自分のレベルを出力する。

## 入力フォーマット

```
N L
x_1 .. x_N   （相手のレベル、N 行）
```

## 制約

- 1 ≦ N ≦ 100,000、1 ≦ L, x_i ≦ 10,000

## 入出力例

入力:
```
3 10
4
20
8
```
出力:
```
3
```
（4 に勝ち +2 → 12、20 に負け → 6、8 に負け → 3）

入力:
```
2 8
8
8
```
出力:
```
8
```
（引き分けが続きレベル変化なし）

## 解法メモ

- 状態は「自分のレベル」1 つだけ。相手を順に走査し、勝ち・負け・引き分けの規則どおり更新する。
- 引き分けは「どちらの分岐にも入らない」ことで自然に表現できる（else 不要）。
- 半分は `Math.floor(x / 2)` で切り捨て。計算量 O(N)。
