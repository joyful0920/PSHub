- **カテゴリ**: 標準入力 / 条件付き合計

## 問題概要

一列に並んだ N 台の車の車間距離（N − 1 個）が与えられる。車間距離が **M メートル以下**の
区間を渋滞と定義したとき、渋滞区間の合計が何メートルかを出力する。車体の長さは無視する。

## 入力フォーマット

```
N M
A_1 .. A_{N-1}   ← 車間距離（N − 1 行）
```

## 制約

- 1 ≦ N ≦ 1000、1 ≦ M ≦ 100、1 ≦ A_i ≦ 200

## 入出力例

入力:
```
4 8
2
10
8
```
出力:
```
10
```
（8 以下は 2 と 8 → 合計 10）

入力:
```
3 6
6
7
```
出力:
```
6
```
（「以下」なので 6 ちょうども含む）

## 解法メモ

- 各車間距離を走査し、`gap <= m` のものだけ合計する。計算量 O(N)。
- **「以下」は M ちょうどを含む**（未満なら含まない）。境界の言葉は要件確認のポイント。
- 車間距離は車 N 台に対して **N − 1 個**。`slice(1, n)` と切り出す（off-by-one 注意）。
