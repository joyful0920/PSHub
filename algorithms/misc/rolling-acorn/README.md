- **カテゴリ**: 標準入力 / シミュレーション（累積和）

## 問題概要

どんぐりが坂道を転がる。坂道は下り坂と上り坂が交互に N 個続く。下った高さの分だけ
次の坂を上れる（空気抵抗なし）。最後まで転がり切れば池に落ちて `YES`、途中で
エネルギー切れなら `NO` を出力する。

各坂の高さ `h_i` は、下り坂なら `≤ 0`、上り坂なら `> 0`（奇数番目が下り、偶数番目が上り）。

## 入力フォーマット

```
N
h_1 h_2 ... h_N
```

## 制約

- 1 ≦ N ≦ 100、-100 ≦ h_i ≦ 100

## 入出力例

入力:
```
5
-10 8 -2 4 -3
```
出力:
```
YES
```
（累積: -10, -2, -4, 0, -3。一度も 0 を超えないので転がり切れる）

入力:
```
4
-10 10 -1 2
```
出力:
```
NO
```
（累積: -10, 0, -1, 1。最後に 1 > 0 でエネルギー切れ）

## 解法メモ

- 「残りエネルギー」を直接持たなくても、**高さの累積和**で判定できる。下りで減り、
  上りで増える累積和が **一度でも 0 を超えたら NO**（下った分より多く上ろうとした）。
- `sum` を 0 から始め「足してから検査」すると、最初の要素の特別扱いが不要になる。
- 累積和がちょうど 0 は OK（エネルギーを使い切っただけ）。計算量 O(N)。
