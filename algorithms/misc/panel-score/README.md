- **カテゴリ**: 標準入力 / 2 次元グリッド（並行グリッド集計）

## 問題概要

H × W のパネルがあり、各パネルに得点が書かれている。ゲーム終了時の各パネルの状態が
`o`（撃ち抜かれた）/ `x`（残っている）で与えられるので、撃ち抜かれたパネルの得点の
合計を出力する。

## 入力フォーマット

```
H W
s_1 .. s_H          ← o/x からなる長さ W の文字列（H 行）
p_{i,1} .. p_{i,W}  ← 得点（H 行）
```

## 制約

- 1 ≦ H, W ≦ 100、1 ≦ 得点 ≦ 100

## 入出力例

入力:
```
2 2
xo
ox
3 4
5 6
```
出力:
```
9
```
（o の位置 = (1,2) と (2,1) → 4 + 5）

入力:
```
1 3
oxo
2 5 8
```
出力:
```
10
```
（o の位置 = (1,1) と (1,3) → 2 + 8）

## 解法メモ

- **状態グリッドと得点グリッドを同じ添字 (i, j) で対応させる**並行グリッドのパターン。
  入力は多段構成なので、状態は `slice(1, h+1)`、得点は `slice(h+1, h+1+h)` と範囲を
  明示して切り出す。
- 二重ループで全マスを走査し、`o` のマスだけ得点を合計する。計算量 O(H×W)。
