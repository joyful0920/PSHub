- **カテゴリ**: 標準入力 / ソート・順位付け

## 問題概要

自分（先頭 index 0）とクラスメイト N 人、計 N+1 人の誕生日 `月 日` が与えられる。
「1 月 1 日から数えて何番目に誕生日を迎えるか」と「4 月 2 日から数えて何番目か」の
2 つの順位を、それぞれ改行区切りで出力する。自分の誕生日は他の誰とも一致しない。

## 入力フォーマット

```
N
M_0 D_0   ← 自分
M_1 D_1
...
M_N D_N
```

## 制約

- 1 ≦ N ≦ 1000、月日は実在する範囲（2 月は 29 日まで）

## 入出力例

入力:
```
3
6 15
2 1
6 10
11 20
```
出力:
```
3
2
```
- 1/1 起点の順: 2/1 → 6/10 → 6/15(自分) → 11/20。自分は **3 番目**。
- 4/2 起点の順: 6/10 → 6/15(自分) → 11/20 → 2/1(翌年扱い)。自分は **2 番目**。

## 解法メモ

- `(月, 日)` を **`月 * 100 + 日`** に符号化すると、その大小がそのまま日付の前後になる
  （月が 100 の位なので日より必ず優先される）。day-of-year の合計と違い 2/29 と 3/1 の
  衝突も起きない。
- 順位 = 「自分より小さい人数 + 1」。自分自身は同じ値なので `< me` に入らず、除外不要。
- 4 月 2 日起点は「4/2 未満の日付を 1 年後ろに回す」= `key < 402` なら `+1300` して末尾へ送る。
  最大 key は 1231 なので、+1300 したものは必ず後ろに来る。計算量 O(N)。
