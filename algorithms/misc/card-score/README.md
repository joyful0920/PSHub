- **カテゴリ**: 標準入力 / 条件集計

## 問題概要

N 枚のカードが与えられる。カードは 3 種類：数値の「点数カード」、`"0"` の「0 点カード」、
`"x10"` の「10 倍カード」。総合点を次のルールで計算して出力する。

1. `"0"` カードがある場合、点数カードのうち最大のもの 1 枚を 0 点にする。
2. 点数カードの合計を求める。
3. `"x10"` カードがある場合、合計を 10 倍する。

「0 点カード」「10 倍カード」はそれぞれ最大 1 枚。点数カードは必ず 1 枚以上ある。

## 制約

- 1 ≦ N ≦ 100、点数カードは 0 ≦ 値 ≦ 100

## 入出力例

入力:
```
5
4 8 0 x10 2
```
出力:
```
60
```
（点数カード 4+8+2=14、`0` あり → 最大 8 を除いて 6、`x10` あり → 60）

入力:
```
3
3 9 5
```
出力:
```
17
```
（`0` も `x10` もないので合計 3+9+5=17 のまま）

## 解法メモ

- カードを 1 回走査して「合計 `sum`」「最大点数カード `maxCard`」「0 点カードの有無」
  「10 倍カードの有無」を集める。
- 適用順（最大を 0 点に → 10 倍）はルールで決まっているので、ループの外で順に処理する
  と優先順位がコードに素直に表れる。計算量 O(N)。
