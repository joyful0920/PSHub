- **カテゴリ**: 標準入力 / 2 次元配列

## 問題概要

N 人による総当たりリーグ戦（引き分けなし）。各試合の勝者・敗者が与えられるので、
`i 行 j 列` に「参加者 i から見た j との結果」を書いた表を出力する。勝ちは `W`、負けは `L`、
自分自身との対戦（対角線 i = j）は `-`。

## 入力フォーマット

```
N
f_1 s_1   ← 勝者 敗者（M = N(N-1)/2 行）
...
```

## 制約

- 2 ≦ N ≦ 20、各組み合わせはちょうど 1 回ずつ

## 入出力例

入力:
```
3
1 3
2 1
2 3
```
出力:
```
- L W
W - W
L L -
```

入力:
```
2
2 1
```
出力:
```
- L
W -
```

## 解法メモ

- N×N の表を `"-"` で初期化。**2 次元配列は `Array.from({length:N}, () => new Array(N).fill("-"))`**
  のようにコールバックで行ごとに新しい配列を作る（`new Array(N).fill(new Array(...))` だと
  全行が同じ参照になりバグる）。
- 1 試合で 2 マスが確定するので、`table[w][l] = "W"` と `table[l][w] = "L"` を対称に埋める。
- 対角線は初期値 `"-"` のまま残る。計算量 O(N^2)。
