- **カテゴリ**: 標準入力 / 集計

## 問題概要

N 日間の旅程が与えられる。各日は「出発地の現地時刻 s 時に出発 → f 時間のフライト →
到着地の現地時刻 t 時に到着」。その日の長さ（出発地の 0 時から到着地の翌 0 時まで）は
`s + f + (24 - t)` 時間になる。N 日間で最短と最長の日の長さを 1 行ずつ出力する。

## 制約

- 1 ≦ N ≦ 100、1 ≦ s, f, t ≦ 23

## 入出力例

入力:
```
2
6 3 9
2 4 22
```
出力:
```
8
24
```

入力:
```
1
5 5 10
```
出力:
```
24
24
```

## 解法メモ

- 各日 `s + f + (24 - t)` を求め、最小値と最大値を出力。
- ソートして両端を取る（`sort((a, b) => a - b)` で数値比較）か、`Math.min/Math.max` で集計。計算量 O(N log N) / O(N)。
