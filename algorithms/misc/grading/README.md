- **カテゴリ**: 標準入力 / 条件抽出

## 問題概要

学生の成績は `テストの点数 a - 欠席回数 b × 5`。ただしマイナスなら 0 とする。
`N` 人分のデータ（学籍番号 1 から順）が与えられるので、成績が合格点 `M` 以上の学生の
学籍番号を順に 1 行ずつ出力する。

## 制約

- 1 ≦ N ≦ 100、0 ≦ M ≦ 100
- 0 ≦ a ≦ 100、0 ≦ b ≦ 15

## 入出力例

入力:
```
3 20
80 5
50 4
10 0
```
出力:
```
1
2
```
（成績 80-25=55 / 50-20=30 / 10-0=10 → 20 以上は 1 番と 2 番）

入力:
```
2 0
3 10
0 0
```
出力:
```
1
2
```
（合格点 0 なので両方合格）

## 解法メモ

- 合格点 `M === 0` は全員合格なので、点数を読まずにループ外で早期リターンできる
  （この判定はループ不変式なので中に置くと N 回無駄に評価される）。
- `M > 0` が確定した後は、成績がマイナスなら必ず不合格。よって 0 クランプ
  （`Math.max(a - 5b, 0)`）は不要で、`a - 5b >= m` だけで判定できる。計算量 O(N)。
