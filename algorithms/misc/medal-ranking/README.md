- **カテゴリ**: 標準入力 / 多重キーソート

## 問題概要

各国のメダル獲得数（金・銀・銅）のリストが与えられる。次のルールで順位付けして上位から出力する。

1. 金の多い方が上位
2. 金が同数なら銀の多い方が上位
3. 金・銀とも同数なら銅の多い方が上位

同着は区別しなくてよい（獲得数を並べ替えて出力するだけ）。

## 制約

- 1 ≦ N ≦ 100、0 ≦ 各メダル数 ≦ 100

## 入出力例

入力:
```
3
5 3 1
5 4 0
8 0 0
```
出力:
```
8 0 0
5 4 0
5 3 1
```
（金 8 が最上位。金 5 同士は銀で比較 → 4 > 3）

入力:
```
2
2 2 2
2 2 2
```
出力:
```
2 2 2
2 2 2
```

## 解法メモ

- 多重キーの降順ソートは `||` チェーンが定石：`b[0]-a[0] || b[1]-a[1] || b[2]-a[2]`。
  前のキーが同点（差 0 = falsy）なら次のキーの比較にフォールバックする。
- 降順なので `b - a`（昇順は `a - b`）。`sort` はデフォルトが辞書順なので比較関数が必須。
- 元配列を壊さないよう `slice()` してからソート。計算量 O(N log N)。
