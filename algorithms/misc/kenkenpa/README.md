- **カテゴリ**: 標準入力 / 連続部分列（スライディングウィンドウ）

## 問題概要

一列に N 個のマスがあり、各マスに丸が 1 個か 2 個書かれている。連続する 3 マスが
「1 個 → 1 個 → 2 個」となっている箇所の数を出力する（重なりは別々に数える）。

## 入力フォーマット

```
N
m_1 m_2 ... m_N
```

## 制約

- 3 ≦ N ≦ 100、1 ≦ m_i ≦ 2

## 入出力例

入力:
```
5
1 1 2 1 1
```
出力:
```
1
```
（先頭の 1,1,2 の 1 箇所）

入力:
```
7
1 1 2 1 1 2 3
```
出力:
```
2
```
（位置 0 と 3 の 2 箇所）

## 解法メモ

- 連続 3 マスのパターン一致を数えるスライディングウィンドウ。
- ループは `i < n - 2` まで回すと、`i+2` が常に範囲内で**境界安全**（off-by-one 回避）。
- `marks[i]===1 && marks[i+1]===1 && marks[i+2]===2` で判定。重なりを許すので位置ごとに数える。
  計算量 O(N)。
