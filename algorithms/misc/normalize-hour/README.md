- **カテゴリ**: 標準入力 / 日時パース・フォーマット

## 問題概要

`"MM/dd hh:mm"` 形式の日時が与えられる。深夜帯の表記で "時" が 24 以上になることがあるので、
時を 0〜23 の範囲に直した日時を同じ形式で出力する。時を繰り上げた結果その月に存在しない日に
なっても、月は調整せずそのまま出力する。

## 制約

- MM, dd, hh, mm は 0 埋め 2 桁。1 ≦ 月 ≦ 12、1 ≦ 日 ≦ 31、0 ≦ 時 ≦ 99、0 ≦ 分 ≦ 59
- 存在しない日付が入力・出力されることがある

## 入出力例

入力:
```
03/15 25:10
```
出力:
```
03/16 01:10
```

入力:
```
06/10 50:05
```
出力:
```
06/12 02:05
```
（50 時 = 2 日 + 2 時。月は調整しないので 12 日のまま）

## 解法メモ

- 変換 → 計算 → フォーマットの 3 段階。
  1. `split` で MM/dd/hh/mm に分解して数値化。
  2. 繰り上がる日数 `Math.floor(hh / 24)` を日に足し、時は `hh % 24`。
  3. 各値を `padStart(2, "0")` で 0 埋めして `"MM/dd hh:mm"` に組み立てる。
- 月・分は変化しない。日は繰り上げるが月はまたがない仕様。計算量 O(1)。
