- **カテゴリ**: 標準入力 / 集計（連想配列）

## 問題概要

N 個のプレイヤー ID（整数）が与えられる。最も多く出現する ID を出力する。
最多が複数ある場合は昇順にスペース区切りで全て出力する。

## 入力フォーマット

```
N
a_1 a_2 ... a_N
```

## 制約

- 1 ≦ N ≦ 1000、1 ≦ a_i ≦ 10000

## 入出力例

入力:
```
5
1 1 2 2 3
```
出力:
```
1 2
```
（1 と 2 が 2 回ずつで最多）

入力:
```
5
1 2 3 1 1
```
出力:
```
1
```
（1 が 3 回で最多）

## 解法メモ

- ID → 出現回数を `Record` で集計（`count[id] = (count[id] ?? 0) + 1`）。
- 最大回数 `Math.max(...Object.values(count))` を求め、その回数の ID を `filter` で全部拾う。
- **落とし穴**: `Object.keys` は文字列なので、そのまま `sort()` すると `"10" < "2"` の辞書順になる。
  `.map(Number)` してから `.sort((a, b) => a - b)` で数値昇順にする。計算量 O(N)。
