- **カテゴリ**: 標準入力 / 集計（連想配列）

## 問題概要

N 人のクラスで投票が行われ、各人が 1 票ずつ名前を書く。最も得票数の多い人の名前を
出力する。最多得票者は 1 人だけ（同数の最多はいない）。

## 入力フォーマット

```
N
a_1       ← 投票された名前（N 行）
...
```

## 制約

- 1 ≦ N ≦ 100、名前は英小文字 1〜10 文字、最多得票者は 1 人

## 入出力例

入力:
```
5
sato
suzuki
takahashi
suzuki
suzuki
```
出力:
```
suzuki
```
（suzuki 3 票が最多）

入力:
```
3
alice
bob
alice
```
出力:
```
alice
```

## 解法メモ

- 名前 → 得票数の対応を `Record<string, number>` で集計。`count[name] = (count[name] ?? 0) + 1`。
  `??`（null 合体）は「未定義なら 0」で、`||` と違い 0 を有効値として残せる（集計の安全な既定）。
- 最多得票者は `Object.entries` を得票降順ソートして先頭。最多が 1 人保証なので同点処理は不要。
- 最大値 1 つだけなら全体ソート O(N log N) の代わりに 1 回走査 O(N) でも取れる。計算量は N 小のため気にしなくてよい。
