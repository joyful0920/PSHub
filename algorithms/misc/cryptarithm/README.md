- **カテゴリ**: 標準入力 / 全探索

## 問題概要

虫食い算の穴 x（十の位、1〜9）と y（一の位、0〜9）を求める。
`a b` が与えられ、次を満たす x, y を出力する（複数あればどれでもよい。無ければ `No`）。

```
(10x + y) × y = 100a + 10x + b
```

つまり「2 桁の数 xy に y を掛けた結果」が、百の位 a・十の位 x・一の位 b の 3 桁の数になる。

## 制約

- 1 ≦ a ≦ 9、0 ≦ b ≦ 9

## 入出力例

| 入力 | 出力 |
| --- | --- |
| `2 4` | `2 8` |
| `5 3` | `No` |

（例1: (10×2+8)×8 = 28×8 = 224 → 百=2, 十=2, 一=4）

## 解法メモ

- x は 1〜9、y は 0〜9 の **90 通りだけ**なので全探索。
- 各組で等式を確認し、最初に成立した組を出力。無ければ `No`。計算量 O(90) = O(1)。
- 代数で解こうとすると y=1 のときのゼロ除算などで面倒 → 探索空間が小さいので全探索が確実。
