- **カテゴリ**: 標準入力 / 集合（種類数で場合分け）

## 問題概要

N 人でじゃんけんをする。各自の手（`rock` / `scissors` / `paper`）が与えられるので、
勝った手を出力する。以下の場合は `draw` を出力する。

- 3 種類すべての手が出ている
- 1 種類の手しか出ていない

勝負がつくのは、ちょうど 2 種類の手が出ているときだけ。

## 入力フォーマット

```
N
a_1 .. a_N   ← 各自の手（N 行）
```

## 制約

- 2 ≦ N ≦ 10、手は rock / scissors / paper のいずれか

## 入出力例

入力:
```
3
paper
rock
paper
```
出力:
```
paper
```
（paper と rock の 2 種類 → paper の勝ち）

入力:
```
2
scissors
scissors
```
出力:
```
draw
```
（1 種類のみ）

## 解法メモ

- 勝敗を決めるのは人数ではなく**手の種類数**。`new Set` で種類を集め、
  `size !== 2` なら引き分け（1 種類・3 種類を 1 つの条件でカバー）。
- 2 種類のときは**勝敗表** `BEATS = { rock: "scissors", ... }` で表引き:
  `BEATS[x] === y ? x : y`（x が y に勝つなら x）。ネストした if 6 分岐が 1 行になる。
- 計算量 O(N)。
