- **カテゴリ**: 標準入力 / シミュレーション（移動距離）

## 問題概要

エレベーターは最初 1 階にいる。停止した階のログが順に N 個与えられるので、
エレベーターが動いた総距離（何階分か）を出力する。

## 入力フォーマット

```
N
f_1 .. f_N   （停止した階、N 行）
```

## 制約

- 1 ≦ N ≦ 100、1 ≦ f_i ≦ 100

## 入出力例

入力:
```
3
5
2
8
```
出力:
```
13
```
（1→5 が 4、5→2 が 3、2→8 が 6。合計 13）

入力:
```
2
3
3
```
出力:
```
2
```
（1→3 が 2、3→3 が 0。合計 2）

## 解法メモ

- 現在位置 `now`（初期値 1）と累計 `total` の 2 状態で走査。
- 各移動距離は方向によらないので **絶対値** `Math.abs(floor - now)` で足す。
- 距離を足したら `now = floor` で位置を更新。計算量 O(N)。
