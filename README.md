## 🎯 algorithms の構成

トピック別にディレクトリを作り、問題ごとに解答を保存します。

```
algorithms/<topic>/<problem>/
├── README.md   # 問題の要約・解法メモ
└── main.ts     # 解答コード
```

TypeScript の練習環境（実行・型チェック・テスト）は [algorithms/README.md](algorithms/README.md) を参照。
